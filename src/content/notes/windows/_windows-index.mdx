## Flashing Windows ISO to USB on Mac OS from CLI 

[Make windows bootable use macos](https://www.uubyte.com/make-windows-10-bootable-usb-on-macos-big-sur.html)

1. download windows{x} iso
2. attach usb drive 


### List USB drives

```sh 
diskutil list external 
```

#### Example 

```sh 
diskutil list external 
# /dev/disk2
```

### Option One 

```sh 
sudo dd if=/path/to/windows.iso of=/path/to/hardware bs=1m
```

#### Example 

```sh 
sudo dd if=/path/to/windows.iso of=/dev/disk2 bs=1m status=progress conv=fsync

```

### Option Two

#### Format the USB drive to MS-DOS

```sh 
diskutil eraseDisk MS-DOS "WINDOWS10" GPT disk2
```

### Remove USB/hard drive 

```sh 
diskutil eject /dev/disk2
```
