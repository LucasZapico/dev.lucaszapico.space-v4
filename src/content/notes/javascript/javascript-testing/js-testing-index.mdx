## Jest 

### Example Single Test

```js:title="foo.js"
export function foo(a,b){
	return a + b
}
```

```js:title="foo.test.js"
import foo from "./foo.js"

test('test foo', async () => {
	expect(foo(3, 6)).toBe(9);
});
```

### Test Specific Path 

```shell
jest path/to/my-test.js
```

or 

```shell
npm test -- path/to/my-test.js
```
