```js
new Array(3).fill(0)
```

### Push Method

```js

const LEN = 3 
const arr = []
for (let i=0; i<LEN; i++){
  arr.push(0)
}
// [0,0,0]
```

###  Mapping method
> Personally I think the mapping methods are the cleanest and most versatile 

```js
const arr = Array.from({length: 3}, () => 0)
// [0, 0, 0]

```

```js
const arr = Array.from({length: 3}, (x,i) => i)
// [0, 1, 2]
```
