## Reference for Common Challenge and Solutions

### Solving Git Ignoring GitIgnore

```shell
git rm -r --cached .
git add .
git commit -m "fixed untracked files"

```

### Revert a Committed Change with reset HEAD

[How to Git Reset to HEAD - devconnected](https://devconnected.com/how-to-git-reset-to-head)




Using the `--hard` flag will remove all changes in the working directory and index. While using the `--soft` flag will keep the changes in the working directory and index.

#### Go Back Commits Without loosing Changes 

```sh
git reset --soft HEAD~<number of commits> 
```

### Go Back Commits and Remove Changes 

#### Discard all current changes 

```sh
git reset --hard HEAD
```

#### Discard all current changes and go back to a specific commit

Go back to single commit before `HEAD`

```sh
git reset --hard HEAD^

```

or 

```sh
git reset --hard HEAD~1 
```

Go back two commits before `HEAD`

```sh
git reset --hard HEAD~2
```

```sh 
git reset HEAD~<number of commits> --hard 
```

### Delete Local and Remote Branches

```shell
// delete branch locally
git branch -d localBranchName

// delete branch remotely
git push origin --delete remoteBranchName
```
