## TroubleShooting Cheat Sheet 

List the image and container names with their associated ports in a clean table

```sh 
docker ps --format 'table{{.Image}}\t {{.Ports}}'
```

```sh
docker ps --format 'table{{.Image}}\t{{.Names}}\t{{.Ports}}'
```
