### General  

#### List All Users

```sh 
vim /etc/passwd
```

#### See Groups of User

```sh 
groups
```

#### Switch from User To Root 

From a given user;

```sh 
sudo -i 
```

#### Switch from Root to User


```sh
su <user-name>
```





## Adding Users, Passwords and Groups 

### Add User

```sh
useradd <new-username>
```

#### Add Password to User 

```sh 
sudo passwd <new-username>
# props for password and password confirmation
```

#### Add USer to Group 

```sh 
sudo usermod -aG <groupname> <username>
```

```sh 
sudo usermod -aG wheel sysadmin

sudo usermod -aG sudo sysadmin
```



#### Delete User 

```sh
deluser -r <user-name>
```

### Change Permissions 

| | | 
|--| --| --| 
| `u` | user | 
| `g` | group | 
| `o` | other people (everyone else) | 

#### Add A Permission 

```sh
chmod <permission-group>+<behavior> <file-or-dir-name>
```

**Example**

```sh 
chmod o+w foo.txt
```

```sh
chmod g+w foo.txt

```

```sh 
chmod o+g foo.txt
```

#### Remove Permission 

```sh 
chmod o-w foo.txt
```


## Set Permissions for group 

```sh 
chmod 754 foo.txt 
# -rwxr-xr--
```
