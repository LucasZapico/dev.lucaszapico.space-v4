# Flash an ISO

1. flash from terminal (cmds the same for linux, and mac)
2. use [Etcher](https://www.balena.io/etcher/) 
    - works on Linux, Windows, and Mac
    - easiest method


## Flash an ISO from term 

### 1. Frind device 

```sh
df # list devices 
```

or

```sh 
lsblk # list all block devices
```
```sh 
lsblk | grep disk # list all disk devices
````

## 2. Unmount device 

```sh
sudo umount /dev/<path-found-in-step-1>
```

### 3. Format device  

```sh
sudo mkrs.<filesystem> /dev/<path-found-in-step-1>
```

#### Example

```sh   
sudo mkfs.ext4 /dev/sdb1
```

### 4. Flash ISO to device 

#### Using `dd`

```sh
sudo dd if=<path/to/iso> of=</dev/path/to/devices>
```
