## Gnome 


### Install Gnome extensions form term

```sh 
sudo apt install gnome-shell-extension-manager

```

```sh 
sudo apt install gnome-shell-extension-gsconnect
```

### List Installed Gnome Extensions 

```sh
gnome-extensions list  
```
results 
```sh
visualizer@sound.org
rectangle@acristoffers.me
clipboard-indicator@tudmotu.com
search-light@icedman.github.com

```

### Disable Gnome Extension 

```sh
gnome-extensions disable <extension-name>
```

**Example** 

```sh
gnome-extnesions disable rectangle@acristoffers.me 
```

### Enable Gnome Extension 

```sh
gnome-extensions enable <extension-name>
```

**Example** 

```sh
gnome-extnesions enable rectangle@acristoffers.me 
```



### Troubleshooting Installing Extensions

Some extensions install processes don't foloow the happy path above. In this case one work around is to download the `gnome-shell-extension-manger` and install it from that ui. 

```sh
sudo apt install gnome-shell-extension-manager 
```

*select browse > search extension > install*

For example to install `dash to panel`

*Gnome Extension Manager -> select Browse -> select Downloads -> enter dash to panel -> select Install*

```sh
kill $(lsof -t -i:8080)
```

## Package Managers 

- `apt` 
- `apt-get`

### `snap`

#### Find Snap Package 

```sh
snap find <package-name>
```

#### Install Snap Package 

```sh
sudo snap install <package-name>
```

#### Remove Snap Package

```sh
sudo snap remove <package-name>
```

#### List Snap packages

```sh
snap list
```

### Window 

From prefix 

`CTRL + b` 

#### Move between windows 

```
CTRL + b <window-prefix>
``` 

**Example**

```
CTRL + b 0
```



#### In Shell Commands

List windows 

```sh
tmux list-window
```

create new window

```sh
tmux new-window
```

split window into two panes 

```sh 
tmux spit-window
tmux spit-window -h // horizontal window
tmux spit-window -v // vertical window
```


### Pane

Kill current pane 

```sh
tmux kill-pane
```

```sh
tmux list-pane
```
