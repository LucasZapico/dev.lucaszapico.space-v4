### Moving and Coping Directories 

#### Move Directories 

The following use the `os` package

```go
import (
  "os"
)
//...
```

**Example**

```go 
//...
func main(){
  source := "/path/to/input"
  dest := "/path/to/dest"

  err := os.Rename(source, dest)
  if err != nill {
    fmt.PrintLn("error moving directory", err)
  }
}
```
