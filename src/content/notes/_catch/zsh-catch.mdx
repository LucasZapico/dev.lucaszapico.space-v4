## Custom Prompt 

[Official Docs - zsh](https://zsh.sourceforge.io/Doc/Release/Prompt-Expansion.html#Prompt-Expansion)


### Syntax Quick Ref 

`%K`
: Promp background color 
: Example - `%K{blue}...`

`%n`
: alias for username

`%m`
: alias for system hostname up to '.'

`%M`
: alias for full hostname

`%B`
: start Boldface mode

`%F`
