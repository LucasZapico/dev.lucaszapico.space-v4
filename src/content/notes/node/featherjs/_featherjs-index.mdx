## Getting Started 

Navigate to project dir and run the following command to create a new featherjs project;

```sh 
npm create feathers@latest <app-name>
```

Alternatively, you can use the featherjs cli to create a new project;

```sh 
npm i -g @feathersjs/cli
```

```sh 
feathers generate app
```
