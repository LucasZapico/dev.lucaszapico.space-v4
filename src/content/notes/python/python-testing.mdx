### Simple Usage 

```python:title="factorial"
def factorial(n):
    if n < 0:
        return "Invalid input"
    elif n in [0, 1]:
        return 1
    else:
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result 
```

  factorial(n)
