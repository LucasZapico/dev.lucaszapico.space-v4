## Cheatsheet 

### Remove Nan 

```python 
# remove any row with nan
df.dropna()
# remove row only when all values are nan
df.dropna(how='all')
# remove row only when specific column values are nan
df.dropna(subset=['column_one', 'column_two'])
```

```python  
df.dropna(how='all', inplace=True)
```

The `inplace=True` argument will modify the original DataFrame.

#### Remove Duplicates 

```python 
# remove any row with duplicates
df.drop_duplicates()
```


### Export 

```python
# export to csv
df.to_csv('file_name.csv')
```
