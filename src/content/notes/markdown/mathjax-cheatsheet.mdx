Let $f(x)$ be a function defined on an interval that contains $x=a$, except possibly at $x=a$. Then we say that,

$$
lim_{x→a}f(x)=L
$$

if for every number $ε>0$
there is some number $δ>0$

such that
$|f(x)−L|<ε$ whenever $0<|x−a|<δ$
