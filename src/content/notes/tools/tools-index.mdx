### Eza

### Bat
```
brew install bat
```
**Description:** Bat is a `cat` clone with syntax highlighting and Git integration. It makes it easier to read code and text files in the terminal by highlighting the syntax and displaying Git status information.
[GitHub Repo](https://github.com/sharkdp/bat)

## Network 

### gping
```
brew install gping
```
**Description:** gping is a visual ping tool that displays network latency in a graphical format. It allows you to see the path that packets take to reach a destination, along with the latency at each hop. This helps diagnose network connectivity issues and identify bottlenecks.

[GitHub Repo](https://github.com/schollz/gping)

### Trippy
```
brew install trippy
```
**Description:** Trippy is a network diagnostics tool that traces the route packets take across the internet, displaying the path visually on a map. It visualizes network hops, geographic locations, and provides detailed information about each hop, helping you understand network performance and identify potential issues.

[GitHub Repo](https://github.com/trippytk/trippy)

**wavemon**

```
sudo apt install wavemon # Debian/Ubuntu
```

**Description:** Wavemon is a command-line based wireless network monitor for Linux. It allows you to view detailed information about your wireless connection, including signal strength, noise level, bit rate, encryption status, and more. It's useful for troubleshooting wireless network issues and monitoring the quality of your connection.

[GitHub Repo](https://github.com/JelteJansma/wavemon)

*   **nvtop**
    ```
    sudo apt install nvtop  # Debian/Ubuntu

    ```
